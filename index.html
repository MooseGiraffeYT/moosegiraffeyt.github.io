Projects

Cartridge 64

Cartridge 64 is an upcoming 3D platformer based off mid-to-late 90s collectathon games. You know the drill by now.

Written in C++, using SDL2 and OpenGL.

PSXFunkin

PSXFunkin is a port of Friday Night Funkin to the PlayStation.

SoniCPort

SoniCPort is a port of Sonic the Hedgehog (1991 Sega Genesis / MegaDrive) to C and modern systems. (Currently using SDL2)

CSE2

CSE2 is a decompilation of Cave Story freeware 1.0.0.6.

Cave Story N64

Cave Story N64 is a port of Cave Story freeware 1.0.0.6 to the Nintendo 64.

OrgMaker 3

OrgMaker 3 is a reimplementation of Studio Pixel's 'Organya' music format. Most prominently used in Cave Story.

CuckyVoxel

CuckyVoxel is a blatant Minecraft clone. I made it so I could test writing backend code, especially with OpenGL.

Written in C++, using SDL2 and OpenGL.

Sonic Onset Adventure

Sonic Onset Adventure is an upcoming 3D Sonic the Hedgehog fangame which hopes to recreate the Adventure Formula with physics, controls, and design similar to the Sonic Adventure duology.

Written in C, using SDL2 and OpenGL.
